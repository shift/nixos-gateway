## 🎯 **Core Brand Elements**

### **Logo System**

#### **Primary Logo**
```
[NIXOS] Gateway
Framework
```
- **Typography**: Inter font family
- **Layout**: Horizontal stacked format
- **Colors**: Primary blue (#2563eb) with secondary green (#10b981)
- **Usage**: Primary brand identification

#### **Logo Variations**
- **Horizontal**: [NIXOS] Gateway Framework
- **Vertical**: [NIXOS] stacked above Gateway Framework
- **Icon**: NG monogram in circular container
- **Wordmark**: Text-only version

#### **Logo Clear Space**
- **Minimum Clear Space**: 0.5x logo height on all sides
- **Minimum Size**: 40px width for digital, 20mm for print
- **Background**: Always use on clean, contrasting background

### **Color Palette**

#### **Primary Colors**
- **Primary Blue**: #2563eb (RGB: 37, 99, 235)
- **Secondary Green**: #10b981 (RGB: 16, 185, 129)
- **Accent Orange**: #f59e0b (RGB: 245, 158, 11)
- **Neutral Dark**: #1f2937 (RGB: 31, 41, 55)

#### **Extended Palette**
- **Light Blue**: #3b82f6 (RGB: 59, 130, 246)
- **Dark Blue**: #1e40af (RGB: 30, 64, 175)
- **Light Green**: #34d399 (RGB: 52, 211, 153)
- **Dark Green**: #059669 (RGB: 5, 150, 105)
- **Light Gray**: #f3f4f6 (RGB: 243, 244, 246)
- **Medium Gray**: #6b7280 (RGB: 107, 114, 128)
- **Dark Gray**: #374151 (RGB: 55, 65, 81)

#### **Usage Guidelines**
- **Primary**: Use primary blue for main brand elements
- **Secondary**: Use green for success states and positive actions
- **Accent**: Use orange for highlights and call-to-actions
- **Neutral**: Use grays for text and backgrounds

### **Typography**

#### **Primary Typeface**
- **Inter**: Modern, clean, highly readable sans-serif
- **Weights**: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
- **Usage**: Headlines, body text, UI elements

#### **Secondary Typeface**
- **JetBrains Mono**: Monospace for code and technical content
- **Weights**: 400 (Regular), 500 (Medium), 600 (SemiBold)
- **Usage**: Code snippets, technical documentation, terminal output

#### **Typography Scale**
- **H1**: 48px / 1.2 (Inter Bold)
- **H2**: 36px / 1.3 (Inter SemiBold)
- **H3**: 24px / 1.4 (Inter Medium)
- **H4**: 20px / 1.4 (Inter Medium)
- **Body Large**: 18px / 1.5 (Inter Regular)
- **Body**: 16px / 1.5 (Inter Regular)
- **Body Small**: 14px / 1.5 (Inter Regular)
- **Caption**: 12px / 1.4 (Inter Regular)

## 🎬 **Video Brand Guidelines**

### **Video Style**

#### **Visual Style**
- **Clean Interface**: Minimal distractions, focus on content
- **Consistent Branding**: Logo placement, color usage, typography
- **Professional Quality**: High resolution, good lighting, clear audio
- **Technical Accuracy**: Accurate representations of technology

#### **Motion Graphics**
- **Smooth Transitions**: Consistent animation style
- **Brand Colors**: Use primary color palette
- **Typography**: Consistent with brand guidelines
- **Pacing**: Appropriate speed for technical content

#### **Screen Recording**
- **High Quality**: 4K resolution when possible
- **Smooth Cursor**: Natural cursor movement
- **Code Highlighting**: Consistent syntax highlighting
- **UI Elements**: Clear, readable interface elements

### **Audio Guidelines**

#### **Voice-over**
- **Professional Quality**: Clear, professional recording
- **Consistent Style**: Knowledgeable but approachable tone
- **Pacing**: Moderate speed with clear enunciation
- **Technical Accuracy**: Correct pronunciation of technical terms

#### **Background Music**
- **Style**: Modern, upbeat, tech-focused
- **Volume**: Subtle, doesn't overpower voice-over
- **Consistency**: Similar style across all videos
- **Licensing**: Royalty-free or properly licensed

#### **Sound Effects**
- **Purposeful**: Used to enhance understanding
- **Subtle**: Not distracting from content
- **Consistent**: Similar sounds for similar actions
- **Professional**: High quality, not cheesy

## 📋 **Brand Compliance**

### **Do's**
- Use official logo files
- Maintain clear space around logo
- Use brand colors consistently
- Follow typography guidelines
- Ensure high-quality production

### **Don'ts**
- Modify or distort logo
- Use outdated logo versions
- Use unapproved colors
- Mix incompatible fonts
- Use low-quality assets
- Misrepresent technical capabilities

### **Quality Standards**
- **Resolution**: Minimum 1080p for video

- **Color Accuracy**: Consistent brand colors
